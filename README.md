## Getting Started

1. checkout to the target version that you want; each version has different features.

```bash
git checkout v1.0.0
```

2. install dependencies:

```bash
npm i
#or
yarn
```

3. run the development server:

```bash
npm run dev
# or
yarn dev
```

After that, you can start to build your own Application!
